Imobiliária

SaaS focada na venda de imovéis

Stack: PHP|Laravel|HTML|Mysql|Redis|CSS|Bootstrap|Js

Sobre:A ideia do projeto é permitir que vendedores, corretores e fotógrafos consigam executar o processo de venda ou aluguel de imóveis através da plataforma de forma dinâmica. Por exemplo: uma pessoa indica o endereço de um imóvel na plataforma; um corretor assume o caso, analisa e realiza os processos necessários; em seguida, um fotógrafo inicia a próxima etapa, e o imóvel fica disponível para venda. Ao final do processo, em caso de venda, todos recebem uma porcentagem proporcional.

Também executei diversas intgrações como:

  • Integração com Google Maps API
  • Integração com Google Places API
  • Integração com Google Geocoding API
  • Integração com Google Auth
  • Integração com outras plataformas de imóveis como Zap Imovéis(via XML)
  • Integração com Bitrix24
